# wonkyufo
A casual side-scrolling single-button steering game

Playable at
https://henriklsp.github.io/wonkyufo/

You do not control the UFO position or speed directly, but the 6th mathematical derivative: position, speed, acceleration, jerk, snap, crackle, pop. Good luck avoiding the asteroids.
Created with Typescript, Vite, Node.js, and Claude Code
